Suppose you want to produce a display like the following.
The problem was posed by James Crippen; the language is Tlingit.

The problem is locating the question/answer indications (Q/A) without
messing up the alignment.

The simplest solution to the problem is:

\codedisplay
\pex[glwidth=5in,textoffset=2.5em]
\a \llap{Q: }\begingl
\gla aad\'oo s\'aw\'e‚ kh\'oo {at lat\'eewu?} //
\glb who Q people.to something=teacher //
\glft `who is the teacher?' //
\endgl
\a \llap{A: }\begingl
\gla J\'aan kh\'oo at~lat\'eewu  \'aw\'a //
\glb John people.to something=teacher COP //
\glft `John is the teacher' //
\endgl
\a\llap{\#A: }\begingl
\gla kh\'oo {at lat\'eewu} \'aw\'e J\'aan //
\glb people.to something=teacher COP John //
\glft `John is the teacher' //
\endgl
\xe
|endcodedisplay
The text offset must be adjusted to accomodate the Q/A marks.
